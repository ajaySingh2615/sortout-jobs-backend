# SortOut Jobs Backend

Spring Boot backend for SortOut Jobs — a job platform (JobHai / Naukri style).

- **Java 21** · **Spring Boot 4** · **PostgreSQL** · **JWT + Spring Security**
- See root [sortout-jobs/README.md](../README.md) for full project overview.

## Run

```bash
docker-compose up -d
./mvnw spring-boot:run
```

Default: `http://localhost:8080`
